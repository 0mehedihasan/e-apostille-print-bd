# e-Apostille Print (Bangladesh)

A Chrome extension that fixes formatting issues in the Bangladesh e-Apostille portal and generates clean, print-ready PDFs.

## ✨ Features

* Fix broken print layout
* Extract full apostille + attached documents
* Preserve signature blocks (Attested / Verified etc.)
* Professional grid layout (3 signatures per row)
* Auto A4 formatting
* Compression modes (Normal / Small / Very Small)
* Auto filename generation

## 📦 Installation (Manual)

1. Download this repository
2. Go to `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the project folder

## 🚀 Usage

1. Open apostille verification page
2. Click **Generate PDF**
3. Save as PDF

## ⚠️ Notes

* Allow popups for `apostille.mygov.bd`
* Works only on official apostille portal
